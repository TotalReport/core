# Feature: Statuses

Statuses are splitted into 3 groups:
1. Passed.
2. Failed.
3. Skipped.

Every group has distinct statuses. The statuses in every group are customizable. It is done to provide more granular results understanding.

The basic embedded statuses:
1. Passed.
    1. Passed.
    2. Passed with warning.
2. Failed.
    1. Failed.
    2. Product bug.
    3. Automation bug.
    4. System issue.
    5. No defect.
    6. To investigate.
3. Skipped.
    1. Skipped.
    2. Aborted.
